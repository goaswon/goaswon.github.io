## SELECT문의 문법
>**SELECT [ 컬럼명 또는 표현식 ]  FROM [ 테이블명, 뷰 명 ] **<br>
> _`*`는 모든 Column을 의미한다._
> 예) SELECT * FROM emp;<br>  emp테이블에서 모든 column을 선택해서 표시해줘

## SELECT문에서 Column을 선택하기

```sql
SELECT [Column의 이름], [Column의 이름], ...
FROM [해당 Column이 있는 TABLE의 이름] ;
```
와 같은 방식으로 선택하면 된다.
